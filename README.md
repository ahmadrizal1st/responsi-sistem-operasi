# Modul 1: Perintah Dasar Sistem Operasi Linux
Modul ini bertujuan agar praktikan mampu menggunakan perintah dasar user dan mengenal format instruksi di Linux.

- whoami: Digunakan untuk melihat nama user yang sedang aktif atau digunakan untuk login saat ini.
- date: Digunakan untuk melihat informasi waktu dan tanggal saat ini dari sistem.

# Modul 2: Sistem File
Modul ini membahas tentang struktur organisasi direktori serta tata cara memanipulasi file dan folder.

- pwd: Singkatan dari Print Working Directory, digunakan untuk menampilkan jalur (path) direktori tempat Anda berada saat ini.
- mkdir: Perintah untuk membuat satu atau lebih direktori/folder baru di dalam sistem file.

# Modul 3: Konfigurasi Linux
Fokus modul ini adalah memahami dan memodifikasi file konfigurasi utama yang ada pada sistem Linux.

- hostname: Menampilkan nama identitas dari mesin atau komputer yang sedang digunakan.
- uname -a: Menampilkan informasi lengkap mengenai kernel sistem operasi, termasuk tipe mesin dan versi sistem.

# Modul 4: Utilitas Linux
Modul ini memperkenalkan berbagai program utilitas untuk membantu pengelolaan sistem, seperti pemantauan sumber daya dan pengarsipan.

- uptime: Digunakan untuk melihat sudah berapa lama sistem menyala sejak booting terakhir.
- free -m: Menampilkan status penggunaan memori (RAM) pada sistem dalam satuan Megabytes (MB).

# Modul 5: Proses Input Output (I/O)
Modul ini menjelaskan konsep standar input, output, serta penggunaan pipa (pipe) untuk menghubungkan satu perintah ke perintah lainnya.

- who | sort: Menggabungkan perintah who (melihat user aktif) dengan sort untuk menampilkan daftar user secara berurutan sesuai alfabet.
- ls | wc -l: Menggunakan pipa untuk menghitung jumlah baris (objek) yang dihasilkan dari daftar perintah ls.

# Modul 6: Manajemen Proses
Modul ini memberikan pemahaman tentang cara menampilkan hubungan antara proses parent dan child serta cara mengontrolnya.

- ps: Singkatan dari Process Status, digunakan untuk melihat kondisi dan daftar proses yang sedang aktif di sistem.
- jobs: Digunakan untuk melihat daftar pekerjaan (jobs) yang sedang berjalan di background atau yang sedang dihentikan sementara (suspended) pada shell saat ini.
