# MENJELASKAN MASING-MASING MATERI DARI MODUL

BAB 1 Perintah Dasar Linux

Pada BAB 1, program menjalankan perintah dasar Linux menggunakan menu berbasis switch case.
Perintah ps digunakan untuk menampilkan daftar proses yang sedang berjalan pada sistem, sehingga pengguna dapat mengetahui aktivitas proses yang aktif.
Perintah pwd digunakan untuk menampilkan direktori kerja saat ini.
Kedua perintah ini termasuk perintah dasar Linux yang sering digunakan untuk memahami kondisi awal sistem dan posisi direktori pengguna. 

BAB 2 Sistem File

Pada BAB 2, program memfokuskan pada pengelolaan sistem file.
Perintah mkdir digunakan untuk membuat sebuah direktori baru sebagai latihan pembuatan folder di Linux.
Perintah ls digunakan untuk menampilkan isi direktori sehingga pengguna dapat melihat file atau folder yang tersedia.
Percobaan ini bertujuan untuk memahami struktur direktori dan manajemen file dasar pada sistem Linux.

BAB 3 Konfigurasi Linux

Pada BAB 3, program melakukan simulasi konfigurasi Linux dengan cara yang aman.
Perintah cp /etc/issue /etc/issue.backup digunakan untuk melakukan backup file konfigurasi sebelum dilakukan perubahan, sesuai dengan prosedur pada modul.
Perintah cat /etc/issue digunakan untuk menampilkan isi file konfigurasi tersebut.
Percobaan ini bertujuan untuk memahami pentingnya proses backup serta cara membaca file konfigurasi sistem Linux.

BAB 4 Utilitas Linux

Pada BAB 4, program membahas penggunaan utilitas Linux untuk pengelolaan file.
Perintah gzip digunakan untuk melakukan kompresi file sehingga ukuran file menjadi lebih kecil.
Perintah tar digunakan untuk mengarsipkan file menjadi satu file arsip.
Percobaan ini bertujuan untuk memahami proses kompresi dan pengarsipan file menggunakan utilitas Linux.

BAB 5 Proses Input Output (I/O)

Pada BAB 5, program membahas konsep proses input dan output pada Linux.
Operator > digunakan untuk mengalihkan (redirect) output dari perintah echo ke dalam sebuah file.
Operator >> digunakan untuk menambahkan (append) output ke dalam file tanpa menghapus isi sebelumnya.
Percobaan ini bertujuan untuk memahami konsep standar output serta pembelokan output ke file sesuai dengan modul praktikum.

BAB 6 Manajemen Proses

Pada BAB 6, program membahas manajemen proses pada sistem operasi Linux.
Perintah ps -u digunakan untuk menampilkan status proses berdasarkan pengguna.
Perintah ps -eH digunakan untuk menampilkan hubungan proses parent dan child dalam bentuk hierarki.
Percobaan ini bertujuan untuk memahami cara melihat, memantau, dan menganalisis proses yang berjalan pada sistem Linux.
