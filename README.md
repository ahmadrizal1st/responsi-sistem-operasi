# MENJELASKAN MASING-MASING MATERI DARI MODUL

- BAB 1: <br>
whoami => digunakan untuk cek user yang sedang login<br>
adduser => digunakan untuk menambahkan user baru<br><br>
- BAB 2: <br>
mkdir => digunakan untuk membuat folder directory baru <br>
nano => digunakan untuk menambahkan file baru + mengedit file tersebut <br><br>
- BAB 3: <br>
nano /etc/issue => digunakan untuk menjadi template yg diprint ketika login pertama lagi <br>
nano /etc/motd => digunakan untuk menjadi template yg diprint ketika sudah selesai login <br><br>
- BAB 4: <br>
gzip -n => digunakan untuk mengkompres file <br>
gunzip => dekompres file <br> <br>
- BAB 5: <br>
cat => digunakan untuk membuka isi konten file tersebut<br>
ps => digunakan untuk melihat proses yg berjalan <br><br>
- BAB 6: <br>
ps -u => digunakan untuk melihat proses yg berjalan di user saat ini <br>
ps -u user => digunakan untuk melihat proses yg berjalan untuk pilihan user secara spesifik <br><br>X
