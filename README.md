BAB 1
whoami digunakan untuk melihat user yang aktif
adduser digunakan untuk menambah user baru

BAB 2 
mkdir digunakan untuk membuat direktori baru
rmdir digunakan untuk menghapus direktori

BAB 3
cp /etc/issue digunakan untuk menyalin file issue yang berisi informasi identitas sistem Linux
-a /etc/update-motd.d digunakan untuk menyalin direktori /etc/update-motd.d beserta seluruh isinya secara lengkap.

BAB 4 
gzip digunakan untuk kompres file
gunzip digunakan untuk dekompresi file

BAB 5
cat digunakan untuk menampilkan isi file secara langsung
ps digunakan untuk menampilkan dafar proses yang sedang berjalan di sistem

BAB 6
pstree digunakan untuk menampilkan semua proses dalam bentuk hirarki sesuai dengan proses parent
ps -a digunakan untuk menampilkan proses yang sedang berjalan yang berhubungan dengan terminal
