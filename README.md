# Bash Script Informasi Sistem

## Deskripsi
Script bash ini digunakan untuk menampilkan informasi dasar sistem.

## Isi Informasi
Script akan menampilkan:
- Nama user yang sedang login
- Tanggal dan waktu saat script dijalankan
- Lokasi direktori saat ini

## Cara Menjalankan
1. Pastikan file memiliki izin eksekusi:
   ```bash
   chmod +x info_sistem.sh
2. Jalankan script:
   ./info_sistem.sh
