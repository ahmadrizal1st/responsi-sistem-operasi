PENJELASAN MASING-MASING MODUL

MODUL 1
* Melihat Isi File
  dengan mengunakan "cat nama_file"
  dimana semua isi dalam file akan ditampilkan
* Mengubah Nama File
  dengan menggunakan "mv nama_file_awal nama_file_baru"
  dimana nama file lama akan berubah nama menjadi yang baru sesuai yang diketikkan

MODUL 2
* Membuat Direktori
  dengan menggunakan "mkdir nama_direktorinya"
  dimana ini akan membuat suatu direktori baru
* Memindahkan Direktori
  dengan menggunakan "mv nama_file nama_direktori/"
  dimana file akan dipindahkan ke dalam direktori tujuan yang ditentukan

MODUL 3
* Konfigurasi File Issue
  dengan menggunakan "/etc/issue"
  dimana file ini digunakan untuk menampilkan informasi sistem saat login dan perubahan memerlukan hak akses sudo
* Konfigurasi File Update
  dengan menggunakan "/etc/update-motd.d/"
  dimana konfigurasi ini digunakan untuk menampilkan pesan sambutan (message of the day) setelah user berhasil login

MODUL 4
* Kompresi File
  dengan menggunakan gzip nama_file
  dimana file akan dikompresi untuk menghemat ruang penyimpanan
* Pencarian File
  dengan menggunakan "find lokasi -name nama_file"
  dimana sistem akan mencari file berdasarkan nama atau pola tertentu

MODUL 5
* File Descriptor (Standard Output)
  dengan menggunakan tanda pembelokan "perintah > file_output"
  dimana hasil keluaran perintah akan disimpan ke dalam file
* Pembelokan Error (Standard Error)
  dengan menggunakan "perintah 2> file_error"
  dimana pesan kesalahan (error) akan diarahkan ke file tertentu.

MODUL 6
* Menampilkan Status Proses
  dengan menggunakan perintah "ps"
  dimana sistem akan menampilkan proses yang sedang berjalan.
* Mengontrol Proses
  dengan menggunakan perintah "jobs"
                              "kill"
  dimana proses dapat dijalankan di background dan dihentikan sesuai kebutuhan user.
