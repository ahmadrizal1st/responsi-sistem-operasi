# MENJELASKAN MASING-MASING MATERI DARI MODUL
Responsi Sistem Operasi menggunakan Bash Script pada Linux.

## BAB 1: Perintah Dasar Sistem Operasi Linux
Perintah dasar Linux adalah perintah yang digunakan untuk mengetahui informasi dasar sistem
dan melakukan pengelolaan user.

Contoh perintah:
- `whoami` → menampilkan user yang sedang login
- `adduser` → menambahkan user baru ke dalam sistem

## BAB 2: Sistem File
Sistem file digunakan untuk mengelola direktori dan file pada Linux.

Contoh perintah:
- `mkdir` → membuat direktori baru
- `nano` → membuat dan mengedit file teks

## BAB 3: Konfigurasi Linux
Konfigurasi Linux digunakan untuk mengatur tampilan dan informasi sistem.

Contoh konfigurasi:
- `/etc/issue` → informasi sebelum login
- `/etc/motd` → pesan setelah login

## BAB 4: Utilitas Linux
Utilitas Linux adalah perintah pendukung untuk mengelola file.

Contoh perintah:
- `gzip` → mengompres file
- `gunzip` → mengekstrak file hasil kompresi

## BAB 5: Proses Input Output
Proses input output berkaitan dengan pengolahan data dan monitoring proses.

Contoh perintah:
- `cat` → menampilkan isi file
- `ps` → menampilkan proses yang sedang berjalan

## BAB 6: Manajemen Proses
Manajemen proses digunakan untuk memantau proses berdasarkan user tertentu.

Contoh perintah:
- `ps -u` → menampilkan proses user aktif
- `ps -u username` → menampilkan proses berdasarkan user tertentu
