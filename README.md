# MODUL 1 :Perintah Dasar Linux
- whoami : digunakan untuk mengetahui user yang sedang login di sistem Linux.
- date : digunakan untuk menampilkan tanggal dan waktu sistem saat ini.
Kesimpulan modul:
Modul ini membantu pengguna memahami kondisi dasar sistem sebelum menjalankan perintah lain.

# MODUL 2 :Sistem File
- pwd : menampilkan direktori aktif tempat user sedang bekerja.
- ls : menampilkan daftar file dan folder di direktori tersebut.
Kesimpulan modul:
Modul ini penting agar pengguna tidak salah lokasi saat membuat, mengedit, atau menghapus file.

# MODUL 3 :Konfigurasi Linux
- nano config.txt : digunakan untuk membuat dan mengedit file teks menggunakan text editor nano.
- cat config.txt : digunakan untuk menampilkan isi file ke terminal.
Kesimpulan modul:
Modul ini mengajarkan dasar pengelolaan file konfigurasi tanpa mengubah file sistem yang sensitif.

# MODUL 4 :Utilitas Linux
- mkdir contoh_folder : digunakan untuk membuat direktori baru.
- gzip contoh.txt : digunakan untuk mengompres file agar ukurannya lebih kecil.
Kesimpulan modul:
Modul ini membantu pengguna mengelola file dan folder secara efisien.

# MODUL 5 :Input dan Output
- ls > output.txt : menyimpan hasil perintah ls ke dalam file teks.
- cat output.txt : menampilkan kembali isi file hasil redirection.
Kesimpulan modul:
Modul ini penting untuk pencatatan output perintah, terutama dalam pembuatan laporan dan logging.

# MODUL 6 :Manajemen Proses
- ps : menampilkan proses yang berjalan pada terminal saat ini.
- ps aux | head : menampilkan daftar proses sistem secara ringkas.
Kesimpulan modul:
Modul ini membantu pengguna memahami bagaimana sistem menjalankan banyak proses secara bersamaan.
